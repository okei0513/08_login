## プロダクトの紹介
- 卒制としてサプリ図鑑のアプリを制作予定
- アプリへの前回の課題にログイン画面を追加
- ログイン画面からメニュー画面への移動とログアウト
- 新規登録画面への移動
- セッションを使ってログインしている時のみメニュー画面を読み込めるよう設定
